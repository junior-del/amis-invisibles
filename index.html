⚙️ Configuration Firebase requise (5 min, gratuit)

Pour que l'app fonctionne en ligne, vous devez créer une base de données Firebase gratuite :

  1. Allez sur console.firebase.google.com
  2. Cliquez "Ajouter un projet" → nommez-le amis-invisibles → Continuer
  3. Dans le menu gauche → Realtime Database → "Créer une base de données"
  4. Choisissez "Commencer en mode test" → Activer
  5. Dans Paramètres du projet (⚙️) → "Ajouter une app Web" (icône </>)
  6. Copiez les valeurs apiKey, authDomain, databaseURL, projectId
  7. Collez-les dans le panneau ci-dessous 👇
